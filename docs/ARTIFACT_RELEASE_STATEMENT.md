# Artifact Release Statement

We release an annotation-layer benchmark and reproducibility code for heading-to-timestamp alignment in long-form public speech recordings, primarily municipal and government meetings.

## Released

- Public source manifest with source URLs and release decisions.
- Heading-to-timestamp annotations.
- Test split metadata.
- Prompt templates.
- Retrieval, indexing, evaluation, and significance-testing scripts.
- Sanitized result summaries.
- Environment templates and dependency files.

## Not released

- Raw videos.
- Raw or converted audio files.
- Converted mono WAV files.
- Full ASR transcripts.
- Text-containing chunks.
- ChromaDB database files.
- API keys, Hugging Face tokens, or `.env` files.

## Reconstruction note

This artifact does not include raw media, full transcripts, text-containing chunks, or automated third-party media downloaders. Users are responsible for ensuring that any use of external source material complies with the applicable licenses, platform terms, and copyright rules.
